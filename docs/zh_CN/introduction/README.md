# 简介 (Introduction)

`rhosocial-activerecord` 是一个现代、类型安全且高性能的 Python Active Record 实现，旨在弥合严格的对象导向建模与原始数据库性能之间的鸿沟。

本章将深入探讨使其与众不同的核心概念。

*   **[核心特性之旅](key_features.md)**: 即使你是初学者，也可以通过这个博客系统的例子，快速体验核心功能。
*   **[设计哲学](philosophy.md)**: 为什么我们需要 "渐进式 ORM" (Gradual ORM)？
*   **[对比分析](comparison.md)**: 与 SQLModel, SQLAlchemy 和 Django 的对比。
*   **[架构设计](architecture.md)**: (进阶) 深入了解我们的分层设计。
