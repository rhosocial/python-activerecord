Refactored logging system to a separate module. The new `rhosocial.activerecord.logging` module provides:

- `configure_logging()`: Global logging configuration entry point
- `LoggingManager`: Singleton manager for all ActiveRecord loggers
- `LoggingMixin` and `BackendLoggingMixin`: Reusable logging functionality
- `ActiveRecordFormatter`: Custom formatter with module context

**Breaking change**: `setup_logger()` no longer modifies the root logger. ActiveRecord logs are now isolated by default (`propagate=False`), preventing log pollution in user applications.

Users can now configure logging globally:

```python
from rhosocial.activerecord.logging import configure_logging
import logging

configure_logging(level=logging.INFO)
```
